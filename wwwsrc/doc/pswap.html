{{boilerplate}}

  • ProtoPedia unwraps the prototype realm (well known to web developers) for use by everyone. Diagrams are manipulated in the user interface and API via operations on their underlying prototype structure.
  • Diagrams are built in the interface or by code. Those built by code can be manipulated and extended in the interface.
  • ProtoPedia includes an open repository of visual elements and diagram types, represented by prototype trees. Elements of the repository are accessible in the interface, as well as in the API. JavaScript programmers everywhere can contribute their own visual elements and diagram types to the repository without disturbing the underlying code base.
