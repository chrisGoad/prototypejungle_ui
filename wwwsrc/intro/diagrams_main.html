Tree Diagram

This is one of several types of diagram in the current catalog (and JavaScript programmers can add their own types). Each diagram type comes with its own set of operations.

Click on a node (that is, a circle). Note that among the operations available in the left panel is "Add Child". Try it. A child node will be added under the selected node. Nodes can be dragged around, and replaced with other shapes, such as squares or text boxes - either one-by-one (topbar "replace") or as a group (topbar "replace prototype"). The "reposition subtree" operation can be used at any time to algorithmically position the nodes in the tree in a regular way that avoids overlap. With these operations, a tree like this
