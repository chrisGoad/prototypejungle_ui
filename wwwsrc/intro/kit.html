Kits

A kit is an item that comes with its own set of operations in the interface. Here, you see a tree kit. If you select a node, then the left-hand panel will display, among other things, the option "Add Child" - a specialized operation for this kit. Try it.

Here is a list of kits, each of which comes with a set of instructions. First the various trees, all with the same interface: (clickable)

Now, the family tree, and the sports bracket.

You can also access the kits in in the catalog, where they appear under the tab named "kit". As usual, you can drag the kits from the tab, for addition to whatever else you are constructing.
